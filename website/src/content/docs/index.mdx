## Quick Example

```toml
[mitigation]
interface = "eth0"
mode = "xdp"
threshold_pps = 1_000_000

[wasm]
rules_dir = "./rules"
hot_reload = true
```

```rust
#[no_mangle]
pub extern "C" fn check_packet() {
  if is_syn_flood() {
    emit_verdict(XDP_DROP);
  }
}
```
